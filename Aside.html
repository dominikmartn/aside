Aside 1.5.2

Cleaner speaker detection, tag management, and a consistent cursor.

Fixed

  • A two-person call stays two speakers. One person's voice could still get split into several speakers over a long call. Aside now uses the calendar to know how many people are on a call, so a one-on-one comes back as just you and them.
  • No more stray transcript gibberish. The odd control code (like <|startoftranscript|>) that leaked into the first transcript line is gone.

New

  • Rename and delete tags. Right-click a tag in the library to rename it everywhere or remove it from all meetings. The meetings stay; only the tag changes.

Improved

  • The cursor behaves. Consistent everywhere now: the arrow over buttons, the text cursor only over text, no more flicker.
